# 函数微分算法的比较实验报告
## 一、实验目的
1. 掌握中心差分法和Richardson外推法计算函数导数的原理和实现方法
2. 通过数值实验比较两种数值微分方法的精度特性
3. 理解不同数值微分方法的适用场景和局限性
## 二、实验原理
### 2.1 待分析函数
$$f(x) = 1 + \frac{1}{2}\tanh(2x)$$

### 2.2 中心差分法
- 计算公式：
  $$f'(x)\approx \frac{f(x+h)-f(x-h)}{2h}$$
### 2.3 Richardson外推法
- 计算公式：
  $$D_{i,j} = \frac{4^j D_{i,j-1} - D_{i-1,j-1}}{4^j - 1},\quad D_{i,0}=\frac{f(x+h_i)-f(x-h_i)}{2h_i}$$
## 三、代码实现
（此处简要说明代码实现的主要步骤和或插入代码）
```python
import numpy as np
import matplotlib.pyplot as plt

# 定义待分析函数
def f(x):
    return 1 + 0.5 * np.tanh(2 * x)

# 定义函数的精确导数
def f_prime(x):
    return np.cosh(2 * x) ** (-2)

# 中心差分法
def central_difference(x, h):
    return (f(x + h) - f(x - h)) / (2 * h)

# Richardson外推法
def richardson_extrapolation(x, h0, n):
    D = np.zeros((n, n))
    for i in range(n):
        h = h0 / (2 ** i)
        D[i, 0] = central_difference(x, h)
    for j in range(1, n):
        for i in range(j, n):
            D[i, j] = (4 ** j * D[i, j - 1] - D[i - 1, j - 1]) / (4 ** j - 1)
    return D

# 实验参数设置
x = 1.0
h0 = 0.1
n_steps = 10
step_sizes = np.array([h0 / (2 ** i) for i in range(n_steps)])

# 计算精确导数
exact_derivative = f_prime(x)

# 中心差分法计算导数
central_derivatives = [central_difference(x, h) for h in step_sizes]

# Richardson外推法计算导数
richardson_derivatives = richardson_extrapolation(x, h0, n_steps)[:, -1]

# 绘制导数计算结果对比图
plt.figure(figsize=(12, 6))
plt.plot(step_sizes, [exact_derivative] * n_steps, 'k--', label='Exact Derivative')
plt.plot(step_sizes, central_derivatives, 'ro-', label='Central Difference')
plt.plot(step_sizes, richardson_derivatives, 'bo-', label='Richardson Extrapolation')
plt.xscale('log')
plt.xlabel('Step Size (h)')
plt.ylabel('Derivative Value')
plt.title('Comparison of Derivative Calculation Results')
plt.legend()
plt.grid(True)
plt.show()

# 计算误差
central_errors = np.abs(central_derivatives - exact_derivative)
richardson_errors = np.abs(richardson_derivatives - exact_derivative)

# 绘制中心差分法误差随步长变化的log-log图
plt.figure(figsize=(12, 6))
plt.loglog(step_sizes, central_errors, 'ro-', label='Central Difference Error')
plt.xlabel('Step Size (h) (log scale)')
plt.ylabel('Error (log scale)')
plt.title('Central Difference Error vs Step Size')
plt.legend()
plt.grid(True)
plt.show()

# 绘制不同阶数Richardson外推法误差随步长变化的log-log图
plt.figure(figsize=(12, 6))
for j in range(n_steps):
    richardson_errors_j = np.abs(richardson_extrapolation(x, h0, n_steps)[:, j] - exact_derivative)
    plt.loglog(step_sizes[:len(richardson_errors_j)], richardson_errors_j, label=f'Richardson Order {j}')
plt.xlabel('Step Size (h) (log scale)')
plt.ylabel('Error (log scale)')
plt.title('Richardson Extrapolation Error vs Step Size')
plt.legend()
plt.grid(True)
plt.show()
```
在代码实现过程中，首先定义了待分析函数f(x)及其精确导数f_prime(x)。接着分别编写了实现中心差分法的central_difference函数和实现 Richardson 外推法的richardson_extrapolation函数。在主程序部分，设定了实验所需的参数，包括计算导数的点x、初始步长h0、外推的步数n_steps等。通过这些函数和参数设置，计算出精确导数、中心差分法的导数以及 Richardson 外推法的导数，并绘制了相应的对比图和误差分析图 。

## 五、实验讨论
### 5.1 两种方法的优缺点分析
1. 中心差分法
   - 优点：实现简便，中心差分法的计算过程相对简单，只需要获取函数在x+h和x−h两点处的函数值，通过简单的四则运算就能得到导数的近似值，易于理解和编程实现。
具有一定精度，在步长选择合适的情况下，中心差分法能够达到二阶精度，对于一些对精度要求不是特别高的场景，它可以快速提供较为可靠的导数近似结果。
   - 缺点：精度受限，与 Richardson 外推法相比，中心差分法的精度相对较低。尤其是在步长较大时，其近似结果与真实导数之间的偏差会比较明显，在对精度要求苛刻的应用中可能无法满足需求。
步长敏感，中心差分法的计算结果对步长的选择极为敏感。步长过大，会导致近似效果不佳，误差增大；而步长过小，虽然理论上可以提高精度，但在实际计算中会引入更多的舍入误差，同样影响计算结果的准确性 。
2. Richardson外推法
   - 优点：高精度，Richardson 外推法通过对不同步长的中心差分结果进行巧妙组合，能够显著提高导数的计算精度。从实验结果来看，其收敛速度更快，在相同步长下比中心差分法更接近精确导数，适用于对精度要求极高的计算场景。
灵活性，可以根据具体的计算需求灵活调整外推阶数。在面对不同的函数和精度要求时，能够通过选择合适的阶数来平衡计算精度和计算复杂度，以达到最优的计算效果。

   - 缺点：计算复杂，Richardson 外推法的计算过程较为复杂，它需要先计算多个不同步长下的中心差分结果，然后再进行多次线性组合运算。这不仅增加了计算量，还对计算资源和时间有更高的要求，在计算资源有限的情况下可能不太适用。
初始步长要求严格，该方法对初始步长的选择有一定要求。如果初始步长选择不合理，可能会导致整个计算过程不稳定，影响最终的计算结果。在实际应用中，需要花费一定的精力来确定合适的初始步长 。
